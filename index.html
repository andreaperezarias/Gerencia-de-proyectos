INICIO

Estimado estudiante, el presente O.V.I. fue diseñado por el equipo del curso de Diseño de página web, y habla acerca de las temáticas del curso de Gerencia de proyectos. Este material tiene contenidos de lectura, multimedia que serán útiles para el desarrollo de las actividades y el aprendizaje de la temática a abordar.

Objetivo General


Desarrollar todo lo relacionado con la parte que tiene la realización de la gerencia de proyectos informáticos, esto quiere decir el paso a paso que se requiere para llevarlo a cabo un proyecto.

Objetivos Específicos


1. Analizar cada una de las fases que requiere el proyecto para poder desarrollarlo.
2. Calcular el tiempo que se requiere para desarrollar cada una de las fases del proyecto y así poder entregarlo a tiempo.
3. Registrar las diferentes operaciones que se hacen en el proyecto.
